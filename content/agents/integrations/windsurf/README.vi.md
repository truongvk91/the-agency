# Tích hợp lướt ván buồm 

Tất cả 61 đại lý của Đại lý được hợp nhất thành một tệp `.windsurfrules` duy nhất. 
Các quy tắc **trong phạm vi dự án** — hãy cài đặt chúng từ thư mục gốc dự án của bạn. 

## Cài đặt 

``` bash 
# Chạy từ gốc dự án của bạn 
cd /của bạn/dự án 
/path/to/agency-agents/scripts/install.sh --tool lướt ván 
``` 

## Kích hoạt Đại lý 

Trong Windsurf, hãy tham chiếu tên nhân viên trong lời nhắc của bạn: 

``` 
Sử dụng tác nhân Frontend Developer để xây dựng thành phần này. 
``` 

## Tái tạo 

``` bash 
./scripts/convert.sh --tool lướt ván 
```
